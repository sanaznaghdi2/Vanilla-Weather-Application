# Vanilla Weather Application

A responsive weather application built with vanilla JavaScript, HTML, and CSS. This project allows users to search for weather information for any city around the world and displays current weather conditions as well as a 5-day forecast.

![Weather App Screenshot](https://via.placeholder.com/600x400.png?text=Weather+App+Screenshot)

## Features

- Current weather display including:
  - Temperature
  - Wind speed
  - Humidity
  - "Feels like" temperature
  - Weather condition with icon
- 5-day weather forecast
- Temperature unit conversion (Celsius/Fahrenheit)
- Responsive design that works on mobile, tablet, and desktop
- Geolocation support to show weather at the user's current location
- Clean, modern UI with smooth transitions

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- OpenWeatherMap API
- Bootstrap 5 (for grid and basic styling)
- Font Awesome (for icons)
- Google Fonts

## How to Use

1. Clone this repository:
```
git clone https://github.com/sanaz-naghdi/Vanilla-Weather-Application.git
```

2. Open the `index.html` file in your browser or set up a local server.

3. Get your API key from [OpenWeatherMap](https://openweathermap.org/api) and replace `YOUR_API_KEY` in the `script.js` file.

4. Search for any city by typing its name in the search box.

## Project Structure

- `index.html` - Main HTML document
- `styles.css` - CSS styling
- `script.js` - JavaScript functionality
